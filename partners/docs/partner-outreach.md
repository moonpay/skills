# MoonPay x [Partner] — Agent Skill Collaboration

## What We're Building Together

We're partnering with [Partner] to embed MoonPay's financial infrastructure directly into your product. The result: any AI agent using [Partner] automatically gets wallet creation, crypto funding, cross-chain bridging, and fiat offramp — without ever leaving your interface.

---

## How It Works

The integration is delivered as a **skill file** — a single markdown document that gives AI agents (Claude Code, OpenCode, any LLM-based agent) the knowledge to use MoonPay's CLI alongside your product.

When a user installs the `[partner]-moonpay` skill:

1. They can create a self-custody wallet in seconds (`mp wallet create`)
2. Fund it with fiat or existing crypto (`mp buy`, `mp token bridge`)
3. Use it natively inside [Partner] — same address, same key
4. Cash out earnings back to bank (`mp virtual-account offramp`)

No separate onboarding. No leaving your product. MoonPay is just there.
